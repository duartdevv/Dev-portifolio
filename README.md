# Portifólio - AntonyDuarte

Um portifólio simples, com o intúito de apresentar minhas habilidades com HTML5, CSS3 e JavaScript.

## Caracteristicas

- Layout responsivo, ou seja: funciona muito bem para celular tanto quanto telas maiores.
- Layout bonito, e com vida. Botões animados (simples, usando apenas css), cores bem escolhidas.
- Conta um pouco sobre mim, o que sei fazer e meus objetivos como Dev.

- https://antonyduarte.github.io/Dev-portifolio/

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Instalação das Dependências

```
npm install
```

Para instalar as dependências do projeto, certifique-se de ter o Node.js instalado em sua máquina. Em seguida, execute:

1. Navegue até a pasta do projeto:
```
cd Dev-portifolio
```

2. Instale as dependências:
```
npm install
```

3. Após a instalação, todos os pacotes necessários estarão disponíveis na pasta `node_modules`.
- JavaScript
